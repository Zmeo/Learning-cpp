# 2019-面试

## 浦发银行/字节跳动/数字政通/百度/美团/金山云/欢聚时代/OPPO

每个人项目不同，所以细聊项目的部分就不单独列出了。

每个问题的回答都会从是什么开始答起，所以有的问题看起来简单，其实回答了挺久的。

就本人看来，回答问题最好是结合实际去答，这样面试官深问的时候是有准备的，如果是纯背概念，后面的问题可能架不住。
